# Tutorials

New to OmniAI? Start here. Each tutorial builds a complete, working system and explains the choices along the way. For single-task recipes, see the [How-to guides](../how_to/index.md) instead; for background on the abstractions, the [Conceptual guide](../concepts/index.md).

| Tutorial | What you build |
| --- | --- |
| [Build an Agent](build_an_agent.md) | A tool-calling agent on the `ChatModel` interface: tools, the agent loop, error feedback, and custom graph control flow. |
| [Multi-channel chatbot](multi_channel_chatbot.md) | One graph served over REST, WebSockets, and Discord with guardrails, auth, and interaction logging. |
| [Continuous learning](continuous_learning.md) | The full self-improvement loop: log interactions, train LoRA adapters in the background, gate them with evals, hot-swap with zero downtime. |
